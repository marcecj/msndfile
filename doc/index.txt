msndfile is a suite of MATLAB Mex wrappers of the {libsndfile} C library for
reading and writing audio files.

This project is split between SourceForge.net and github.com, although it is
in the process of moving completely to the latter.  The {githubpage} hosts
this website, the git repository, and a bug tracker, and is basically where
development happens.  The SF.net {projectpage} currently still hosts
downloads.
