msndfile is a suite of MATLAB Mex wrappers of the {libsndfile} C library for
reading and writing audio files.

This project lives at github.com.  Got to the {githubpage} for access to the
git repository, bug tracker, downloads, etc..

The old SF.net {projectpage} is considered obsolete and will be removed in the
future.
