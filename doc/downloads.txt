Download
--------

The most recent version of msndfile is 1.1. You can download the source
package, binaries (32bit Windows and Linux) and a PDF version of this website
here:

Windows::
http://sourceforge.net/projects/msndfile/files/1.1/msndfile-1.1-win32.zip/download
Linux::
http://sourceforge.net/projects/msndfile/files/1.1/msndfile-1.1-glnx86.zip/download
Source::
http://sourceforge.net/projects/msndfile/files/1.1/msndfile-1.1-src.zip/download
Manual::
http://sourceforge.net/projects/msndfile/files/1.1/msndfile-1.1-manual.pdf/download

Older versions can be found at the SourceForge {projectpage}.  The Windows
binaries bundle the libsndfile DLL.  Under Linux and Mac it is expected that
libsndfile is already installed (e.g., via a package manager).
